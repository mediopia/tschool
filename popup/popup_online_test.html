온라인 시험

시험명 2019 과목별 테스트 온라인 시험
시험기간 2019.05.31~2019.06.30
문항수 문항(객관식 20문항, 주관식 0문항)
시험시간 총 60분
    • 시험응시 가능한 횟수는 1회입니다.
    • '제출하기' 버튼을 선택하면 재응시는 불가능합니다.
    • 온라인 시험 성적 확인은 나의 강의실 > 성적/진도에서 '성적표 보기'를 선택하세요.
    • ※ 온라인 시험을 진행하시기 전에 꼭 읽어 주세요.
    • 온라인 시험지는 총 3~4 set 중, 1set가 출제됩니다. (A set, B set, C set 중 한가지가 선택되어 출제됩니다.)
    • 시험 중 마우스 포인터는 시험지 안에서만 위치될 수 있도록 해주세요.
    • 시험지 창 외에 다른 곳으로 마우스 포인터가 이동할 경우 시험이 자동 중단될 수 있으니 마우스 사용에 꼭 주의해 주시기 바랍니다.
